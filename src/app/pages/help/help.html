What is Flowdesk?

Flowdesk is a Kanban-based project management tool. Tasks move through four workflow states: To Do, In Progress, Await Feedback, and Done. This structure helps teams keep work visible, track progress, and focus on priorities.

Account and Access

  • Sign Up: Create a new user account from the signup page.
  • Log In: Access your workspace from the login page.
  • Protected Pages: Board, Dashboard, Add Task, Contacts, and Help require authentication.
  • User Menu: Open the profile button in the header to access Help, Legal Notice, Privacy Policy, and Logout.

Navigation Overview

  • Dashboard: Quick status view, progress, due soon, and overdue tasks.
  • Add Task: Create a new task in a dedicated form.
  • Board: Search, filter, open, move, and update tasks.
  • Contacts: Manage your contact list and open detailed contact cards.
  • Responsive Layout: On mobile, navigation and detail areas are optimized for smaller screens.

Dashboard Features

  • Greeting and Date: Personalized welcome with current day context.
  • Progress Tracking: Completion bar and ratio of finished versus total tasks.
  • KPI Cards: Direct entry points to To Do, In Progress, Await Feedback, Done, Urgent, Overdue, and total tasks.
  • Due Soon and Overdue Lists: Prioritized task lists that can open the related task quickly.

Board Features

  • Task Search: Find tasks by title or description.
  • Compact View: Toggle between display densities for better overview.
  • Advanced Filters: Filter by priority, category, due date, assignees, and "Assigned to me".
  • Quick Reset: Clear all active filters with one click.
  • Board Columns: Tasks are grouped by workflow status and can be moved between columns.
  • Add Task Modal: Create tasks directly from the board.
  • Task Detail Modal: Open a full task view without leaving the board.

Task Management

  • Create Tasks: Enter title, description, category, priority, due date, assignees, and subtasks.
  • Edit Tasks: Update existing task data from the detail view.
  • Delete Tasks: Remove tasks from the project when no longer needed.
  • Subtask Progress: Mark subtasks as done directly in the task detail panel.
  • Visual Priorities: Priority indicators help identify urgent work quickly.

Contacts Management

  • Task Assignment: Assign contacts to tasks for clear responsibility.

Recommended Workflow

  1. Create your account or log in.
  2. Add contacts first so tasks can be assigned immediately.
  3. Create tasks via Add Task or directly from the Board.
  4. Use Dashboard insights to prioritize urgent and overdue work.
  5. Use Board filters and search to focus on specific task groups.
  6. Update subtasks and move cards regularly to keep progress transparent.
